Angewandte Gruppentheorie

<u>Def</u>.: Sei M eine Menge mit einer Verknüpfung f: M x M -> M,
f(a, b) = a ◊ b, so dass
A: (a ◊ b) ◊ c = a ◊ (b ◊ c)
	Assoziativität
N: ∃ e: a ◊ e = a ∀ a (auch e ◊ a = a)
	Neutrales Element
I: ∀ a ∃ !a : a ◊ !a = e (auch !a ◊ a = e)
	Inverses Element

Dann nennt man (M, f) eine Gruppe.

<u>Bsp</u>.: reelle Zahlen und Addition
-> ist assoziativ -> Neutrales Element ist 0 -> inv. ist negativ

<u>Bsp</u>.: reelle Zahlen ≠ 0 und Multiplikation

<u>Bsp</u>.: komplexe Einheitswurzeln (roots of unity, RU)
	Def.: Sei n ∈ {1, 2, 3, ...}, dann nennt man die n (verschiedenen) Nullstellen des Polynoms z<sup>n</sup>-1 die n-ten Einheitswurzeln.
	(Diese bilden die Ecken des regulären Polygons mit n-Ecken, wobei eine Ecke auf <br>z = 1 liegt)

	(Bei komplexen UR nennt man Generator oft <u>primitive Einheitswurzeln</u>, Abk. PRV)
Eine solche Gruppe nennt man zyklisch.

<u>Def</u>.: Für ein Element a, betrachte die Folge a<sup>1</sup>, a<sup>2</sup>, a<sup>3</sup> …, dann nennt man den minimalen Exponenten k ≥ 1, so dass a<sup>k</sup> = 1 ist, die Ordnung von a.

<u>Beweis zyklische Gruppen sind kommutativ:</u>
Nicht kommutativ -> nicht zyklisch!

∃g, so dass jedes Element als g<sup>e</sup> schreiben kann.
Sei a = g<sup>e</sup> und b = g<sup>f</sup>, dann a ◊ b = g<sup>e</sup> ◊ g<sup>f</sup> = g<sup>e + f</sup> = g<sup>f + e</sup> = b ◊ a.

<u>Def</u>.: Sei G = (M, ◊) eine Gruppe, dann nennt man |M| die <u>Ordnung</u> der Gruppe und schreibt |G| dafür.

<u>Def</u>.: Mit G = (M, ◊), sei U ⊆ M, so dass a ◊ b ∈ U (a, b ∈ U),<br>dann nennt (U, ◊) eine <u>Untergruppe</u> von G.

## Definition Gruppe
Was ist überhaupt eine Gruppe?
- algebraische Struktur mit einer Menge G und einer Verknüpfung ◊ => (G, ◊)

(G, ◊) ist nur dann eine Gruppe, wenn:
- **Assoziativ**:
	(a∘b)∘c = a∘(b∘c) => *wir dürfen Klammern beliebig setzen*
- **Neutrales Element**:
	- es muss ein (!) neutrales Element für die Verknüpfung geben
	- => es gibt ein neutrales Element e, das beim Verknüpfen mit einem anderen Element nichts verändert
	- a∘e = a und e∘a = a
	- Beispiel: (ℕ, +) => alle natürlichen Zahlen und Verknüpfung Addition
		- wann ist 2 + x = 2 ?
		- für das neutr. Element 0
- **Inverse Elemente**:
	- In (M, ∘) mit neutralen Element n heißt a<sup>-1</sup> inverses Element von a, wenn:
	- a ∘ a<sup>-1</sup> = a<sup>-1</sup> ∘ a = n
	- Beispiel: (ℤ, +) => wir suchen etwas, so dass das neutrale Element rauskommt
		- 7 + x = das neutrale Element bei der Addition
		- was ist das neutr. Elem. bei der Add.? => wie schon dafür geprüft = 0
		- 7 + x = 0 => 7 + (-7) = 0
		- -5 + x = 0 => -5 + 5 = 0
		- wir haben immer ein inv. Element zu unserem untersuchten Element
	- Beispiel: (ℚ, ∙)
		- 4 ∙ x = 1 => neutr. Element ist 1
		- 4 ∙ 1/4 = 1
	- <font color="#ff0000">Wichtig: Das inv. Element muss teil der Menge sein!</font>
- **Abgeschlossenheit**:
	- Verknüpfung ist nur dann eine Verknüpfung, wenn die Abgeschlossenheit schon gilt => nicht zwingend prüfbar
	- wenn wir zwei Elemente der Gruppe nehmen und diese verknüpfen, dann muss ein Element der Gruppe herauskommen
	- Beispiel: (ℤ, +)
		- 7 + 1 = 8 => 8 ∈ ℤ 

### Beispiel
(ℤ, +) => Menge der ganzen zahlen mit der Operation Addition:
- Assoziativ? => Ja, (a + b) + c = a + (b + c).
- Neutrales Element? => Ja, 0.
- Inverses Element? => Ja für a ist das Inverse -a, denn a + (-a) = 0.

## Zyklische Gruppen
*"Eine Gruppe ist zyklisch, wenn es ein Element gibt (-> heißt Generator), mit dem man alle anderen Elemente der Gruppe erzeugen kann."*
-> dieses Element wird einfach immer wieder "verknüpft" (z.B. addiert oder potenziert)

<u>Beispiel</u>: Eine Uhr mit 12 Stunden
Die Stunden auf einer Uhr bilden eine zyklische Gruppe:
- Die Elemente sind {0, 1, 2, …, 11}.
- Verknüpfung ist hier **Addition mod 12** (-> weil wenn man über 12 "drüberzählt", fängt man wieder bei 0 an).

Der Generator ist dann 1:
- Beginne bei 0, dann addiere immer 1:
	0, 1, 2, 3, 4, … , 11, 0, 1, …

> [!note]- Die "Rechnung"
> Die Gruppe Z<sub>12</sub> besteht aus den Elementen {0, 1, 2, …, 11}, und die Verknüpfung ist <br>+ mod 12.
> 
> **Generator g = 1:**  
> Die Potenzen von g sind:
> 
> - g<sup>0</sup> = 0
> - g<sup>1</sup> = 1
> - g<sup>2</sup> = 2
> - g<sup>3</sup> = 3
> - ...
> - g<sup>11</sup> = 11
> - g<sup>12</sup> (Zurück zum Anfang, weil 12 mod  12 = 0)
> 
> Mit g = 1 kannst du **alle Elemente** der Gruppe erzeugen: {0, 1, 2, …, 11}.

Es wiederholt sich wie ein Kreis -> deswegen zyklisch
	*"Können durch wiederholte Anwendungen eines einzigen Elements (dem Generator) erzeugt werden."*

- zyklische Gruppen sind <span style="background:#d4b106">immer</span> **kommutativ** (die Reihenfolge spielt keine Rolle, <br>z.B. a + b = b + a).

> [!note]- Weiteres Beispiel
> Betrachte die 4. Einheitswurzeln {1, i, -1, -i}, die Nullstellen von z<sup>4</sup> -1 = 0. Die Verknüpfung ist hier die Multiplikation.
> 
> Generator g = i:
> - g<sup>0</sup> = 1
> - g<sup>1</sup> = i
> - g<sup>2</sup> = -1
> - g<sup>3</sup> = -i
> - g<sup>4</sup> = 1 (-> zurück zum neutralen Element)
> 
> Auch hier kannst du alle Gruppenelemente {1, i, -1, -i} durch Potenzen von g = i erzeugen.


## Ordnung
*"Die Ordnung eines Elements in einer Gruppe ist die Anzahl der Schritte, die man braucht, um mit diesem Element das **neutrale Element** zu erreichen."*

Beim Beispiel mit der Uhr:
- neutrales Element ist die 0
- wenn Generator 1 ist, dann:
	1 + 1 + 1 … (12mal und dann ist man wieder bei 0)
	=> die Ordnung von 1 ist also 12.

- ein anderes Element, z.B. 6: 
	- 6 + 6 = 12 mod 12 = 0
	=> Die Ordnung von 6 ist 2, weil es nur 2 Schritte braucht, um 0 zu erreichen.

## Untergruppen
*"Eine Untergruppe ist eine kleine Gruppe, die in der großen Gruppe enthalten ist. Sie muss auch die Gruppeneigenschaften (Assoziativität, Neutrales Element, Inverses Element) erfüllen."*

<u>Beispiel</u>:
Gruppe ℤ<sup>6</sup> = {0, 1, 2, 3, 4, 5}
Verknüpfung: Addition mod 6:
- 0 + 0 = 0
- 1 + 2 = 3
- 4 + 5 = 3 mod 6
- …
Eine mögliche Untergruppe ist U = {0, 3}, weil:
1. Verknüpfung bleibt in der Untergruppe
	3 + 3 = 6 mod 6 = 0 (bleibt in U).
2. Neutrales Element 0 ist in U.
3. Jedes Element hat eine Inverse:
	- Das Inverse von 0 ist 0
	- Das Inverse von 3 ist 3, denn 3 + 3 = 6 mod 6 = 0

*"Die Ordnung jeder Untergruppe teilt die Ordnung von G"*
<u>Was bedeutet das?</u>
Die Ordnung einer Gruppe ist ja die Anzahl der Elemente in der Gruppe.
- Wenn G eine Gruppe ist und U eine Untergruppe von G, dann gilt:
	|U| teilt |G|

=> die Anzahl der Elemente in der Untergruppe U ist immer ein Teiler der Anzahl der Elemente in der großen Gruppe G.

> [!seealso]- Beispiel
> Die Gruppe ℤ<sup>6</sup> {0, 1, 2, 3, 4, 5} hat |G| = 6 Elemente.
> Mögliche Untergruppen:
> 1. U = {0}:
> 	Nur das neutr. Element. Ordnung |U| = 1, und 1 teilt 6.
> 2. U = {0, 3}:
> 	Ordnung |U| = 2, und 2 teil 6.
> 3. U = {0, 2, 4}:
> 	Ordnung |U| = 3, und 3 teilt 6.
> 4. U = {0, 1, 2, 3, 4, 5}:
> 	Das ist die ganze Gruppe. Ordnung |U| = 6, und 6 teilt 6.
> 
> Es gibt keine weiteren Untergruppen, weil nur 1, 2, 3, 6 echte Teiler von 6 sind.

> [!note]- Für den Brainfuck
> <u>Sätzchen</u>:
> *"In der zyklischen Gruppe mit |G| = n gibt es φ(n) primitive Wurzeln, wobei φ(n) die Anzahl der Zahlen e, 1 ≤ e < n ist mit e ⊥ n ("e teilerfremd zu n")"*
> 
> Erklärung:
> 1. Was ist eine zyklische Gruppe?
> Eine zyklische Gruppe ist eine gruppe, bei der es ein Generator-Element g gibt, sodass alle Elemente der Gruppe durch Potenzen (bzw. Iterationen der Verknüpfung) von g erzeugt werden können.
> Für eine Gruppe G mit |G| = n:
> <center>G = {g<sup>0</sup>, g<sup>1</sup>, g<sup>2</sup>, …, g<sup>n - 1</sup>}.</center>
> 2. Was sind primitive Wurzeln?
> Ein Element g ist eine primitive Wurzel (auch Generator genannt), wenn es tatsächlich alle n Elemente der Gruppe durch seine Potenzen erzeugt, ohne Wiederholungen vor <br>g<sup>n</sup> = 1.
> Ander gesagt: g<sup>e</sup> ≠ 1 für 1 ≤ e < n, und g<sup>n</sup> = 1.
> Beispiel: Betrachte ℤ<sub>7</sub> = {0, 1, 2, 3, 4, 5, 6} mit der Multiplikation mod 7:
> - 3 ist eine primitive Wurzel, weil:
> 	3<sup>1</sup> = 3, 3<sup>2</sup> = 9 mod 7 = 2, 3<sup>3</sup> = 27 mod 7 = 6, …, 3<sup>6</sup> = 1.
> 	Es erzeugt alle 6 Elemente der Gruppe.
> 
> 3. Was ist φ(n)?
> φ(n) ist die Euler'sche Phi-Funktion und gibt die Anzahl der Zahlen e in {1, 2, …, n - 1} an, die teilerfremd zu n sind.
> - Zwei Zahlen a und n sind teilerfremd, wenn ggT(a, n) = 1.
> - Beispiel:
> 	Für n = 12 sind die Zahlen 1, 5, 7, 11 teilerfremd zu 12, also φ(12) = 4.
> 
> 4. Wie hängen primitive Wurzeln und φ(n) zusammen?
> In einer zyklischen Gruppe G mit |G| = n:
> - Es gibt genau φ(n) viele primitive Wurzeln.
> - Warum? Weil die primitive Eigenschaft eines Generators g von der Ordnung seiner Potenzen abhängt. <br>Nur die Potenzen g<sup>e</sup> mit ggT(e, n) = 1 (d.h., e ist teilerfremd zu n) können primitive Wurzeln sein.
