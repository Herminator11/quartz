Eigenschaften der Relation (Mit Menge M und a, b, c ∈ M):
1. a = a für alle a Reflexivität (R)
2. a = b => b = a Symmetrie (S)
3. a = b und b = c => a = c Transitivität (T)

Def.: Eine Relation für die R, S, T gilt, nennt man eine <mark style="background: #FFB86CA6;">Äquivalenzrelation</mark>.

# Nochmal in eigenen* Worten
\*teils eigen teils ChatGPT/YouTube Videos<br>
> [!info] Einfach erklärt
> Mit einer Äquivalenzrelation kann man **Beziehungen** zwischen Dingen in einer Menge darstellen. Dabei werden zwei Dinge "äquivalent" genannt, wenn sie **gleiche** oder **ähnliche Eigenschaften** haben.

Eine Äquivalenzrelation kann man an **drei Eigenschaften** erkennen:

1. <font color="#00b050">Reflexivität</font>: Jedes Element der Menge M ist äquivalent <font color="#00b050">zu sich selbst</font>.
	*Beispiel: Alex ist blond, genau wie er selbst, also a ~ a.*<br> (∀ a ∈ M : a ~ a)

2. <font color="#4f81bd">Symmetrie</font>: Wenn ein Element a aus M äquivalent zu b ist, dann ist auch b äquivalent zu a. Eine solche Beziehung beruht also auf <font color="#4f81bd">Gegenseitigkeit</font>.
	*Beispiel: Wenn Alex dieselbe Haarfarbe hat wie Berta (a ~ b), dann hat Berta auch dieselbe Haarfarbe wie Alex (b ~ a).*<br> (∀ a, b ∈ M : a ~ b => b ~ a)

3. <font color="#8064a2">Transitivität</font>: Ist a äquivalent zu b und b wiederum äquivalent zu c, dann ist auch a äquivalent zu c, das heißt, die Beziehung <font color="#8064a2">überträgt sich</font>.
	*Beispiel: Wenn Clara dieselbe Haarfarbe hat wie Berta (c ~ b) und Berta wiederum dieselbe Haarfarbe hat wie Alex (b ~ a), dann kannst du daraus schließen, dass auch Clara und Alex dieselbe Haarfarbe haben: a ~ c.*
	 (∀ a, b, c ∈ M : a ~ b ∧ b ~ c => a ~ c)

> [!note] Definition
> Eine **Äquivalenzrelation R** beschreibt eine <font color="#00b050">reflexive</font>, <font color="#548dd4">symmetrische</font> **und** <font color="#8064a2">transitive</font> **Beziehung** zwischen je **zwei Elementen**. Indem du Elemente zusammenfasst, die paarweise äquivalent zueinander sind, kannst du damit die Grundmenge M in disjunkte Teilmengen **gliedern**. Mathematisch ausgedrückt ist eine Äquivalenzrelation damit eine **Teilmenge** des **kartesischen Produktes M x M** (R ⊂ M x M).

- um zu beweisen, dass es tatsächlich eine Äquivalenzrelation ist, muss man überprüfen, ob sie die drei Eigenschaften <font color="#00b050">Reflexivität</font>, <font color="#4f81bd">Symmetrie</f> und <font color="#8064a2">Transitivität</font> erfüllt.
