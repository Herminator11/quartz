- C++ ist eine statisch typisierte Sprache
	-> d.h. Code wird vom Compiler in Maschinencode übersetzt

- C++ ist eine objektorientierte Programmiersprache 

> [!info]
> Eine **Klasse** definiert und kapselt Eigenschaften und Funktionen . Ein **Objekt** ist eine Instanz einer Klasse.

- **Abstraktion**: Eine Klasse im Code *abstrahiert* ein reales Objekt oder einen realen Akteur, indem es Schnittstellen bereitstellt, die definieren, wie bestimmte Funktionalitäten ausgeführt werden bzw. wie die Klasse mit anderen Klassen kommuniziert.

- **Kapselung**: Zusammengehörige Daten und Methoden werden in Klassen gebündelt und Zugriff auf sie nur über festgelegte Schnittstellen erlaubt.

- **Polymorphie**: Unterschiedliche Klassen können dieselbe Schnittstelle implementieren und auf unterschiedliche Weise auf diese Schnittstelle reagieren.

- **Vererbung**: Klassen können von einander erben und dadurch erweitert bzw. spezialisiert werden.
