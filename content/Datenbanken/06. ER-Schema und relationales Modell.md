## Gruppierung
2 Schritte:
1. Tupel mit gleichen Attributwerten für A<sub>1</sub>, …, A<sub>n</sub> gruppieren
2. auf jede Gruppe Funktionen F<sub>1</sub>, …, F<sub>m</sub> anwenden

- Bsp. für Aggregatfunktionen F:
	- count, sum, avg, min, max

z.B.:

```sql
SELECT Semester, count(*)
FROM Studenten
GROUP BY Semester;
```

### Aggregatfunktionen

-> COUNT( \* | \[DISTINCT] column )
- COUNT(\*): Anzahl Zeilen in Tabelle/Gruppe
- COUNT(column): … mit column ≠ null
- COUNT(DISTINCT column): … u. unterschiedl. Werten

-> AVG, SUM, … ähnlich (Aber ohne \*)
-> null - Werte nicht ignorieren: AVG(NVL(column,0))
-> Verschachtelung möglich z.B.:

```sql
SELECT MAX(AVG(Gehalt))
FROM Mitarbeiter
GROUP BY Abteilung;
```
-> liefert maximales Durchschnittsgehalt!

### Gruppen selektieren
… HAVING condition
-> nur Gruppen auswählen, die condition erfüllen
z.B.:

```sql
SELECT Abteilung, MAX(Gehalt)
FROM Mitarbeiter
GROUP BY Abteilung
HAVING MAX(Gehalt) > 10000;
```
